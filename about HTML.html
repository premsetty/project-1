My understanging about HTML

HTML - Hyper Text Markup Language

It is the standard for creating webpages. It describes the structure of the webpage.

It tells the browser how to display the contents.

It consists of HTML elements which are defined by : <tagname> Content </tagname>

Example: <h1> This is my head </h1>

Some tags are:

<!DOCTYPE html>

<html>

<head>

<title>

<body>

<h1>

<p>
